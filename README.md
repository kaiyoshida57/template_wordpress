# WordPress template
WordPress site template
PHP + Sass + Browser-Sync
# use version:
gulp v4.0.2
node v12.13.1
npm v6.12.1
# install package
$ yarn
or
$ npm init -y
$ npm i
# start local server and watch
$ yarn gulp
or
$ npx gulp
